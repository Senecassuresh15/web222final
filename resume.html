My WEB 222 Experience


I had a great time taking Seneca's WEB222 course. I learned many things I did not know previously, including how web pages were structured, how they were styled, how to implement certain functionalities and considerations that web developers take when designing them. I enjoyed learning about and practicing programming the small front end event handler programs in the last assignment, as well as creating web pages that linked to each other in the third assignment.
However, I struggled with the CSS portion of the class, which overwhelmed me with all the style options and considerations I had to take to get the structure I wanted for my web pages. I think it was partially due to me underestimating how steep the initial learning curve of CSS would be, since the sheer number of concepts and options can be overwhelming. Even as I am creating the web pages for this final assessment, I am having the most difficulty with the CSS part of the page design.
I felt like I struggled with the course load in my second semester of the Seneca Computer Programming and Analysis program, particularly during the C++ focused Object-Oriented Programming course and the Web Development course. In addition, it was a challenging experience to learn how to handle school, work and personal life, and I strugged with time management during the semester. I hope to do better in the future using what I learned this semester.
